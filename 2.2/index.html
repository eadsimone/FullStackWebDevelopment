In this Task, you’ll use built-in Node modules to perform various tasks for your project. You’ll be using the http, url, and fs modules in combination to return files to the user and log requests. Directions Using the terminal, create a “documentation.html” file within your “movie_api” directory. This will contain instructions for other developers on how to use the API you’re building. You can add whatever instructions you think might be helpful, and you'll continue to update this document as you go along. For now, let’s include a title for your API, as well as a basic intro paragraph on what the API does. You can refer back to the project brief at this stage as needed. Within your project directory, create a “server.js” file that imports the http module, listening for requests on port 8080. Now, import the url module at the top of your “server.js” file. Parse the request.url to determine if the URL contains the word “documentation”. If it does, return the “documentation.html” file to the user; otherwise return the “index.html” file. Create a “log.txt” file in your project directory. For all requests coming in to your “server.js” file, use the fs module to log both the request URL and a timestamp to the “log.txt” file. Open GitHub and create a new repository for your project. Push your local project directory to GitHub. For help with how to do this using the terminal, check out GitHub Help: Adding an Existing Project Using the Command Line.
